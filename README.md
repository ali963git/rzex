# 🚀 RZEX — منصة تداول العملات الرقمية (Cryptocurrency Trading Platform)

منصة تداول متكاملة مبنية على نمط **الخدمات المصغّرة (Microservices)**، تدعم المصادقة بخطوتين (2FA)، محفظة رقمية، محرك مطابقة الأوامر، بيانات السوق في الوقت الفعلي، مراقبة أمنية، وواجهة أمامية تفاعلية.

---

## 📌 المميزات الرئيسية

- ⚡ **بوابة API موحدة (Gateway)** مع تحميل طلبات متوازن وسيطرة على الأمان.
- 🔐 **نظام مصادقة متكامل** مع **JWT** و **المصادقة الثنائية (2FA)** عبر TOTP.
- 💰 **خدمة محفظة** تدعم قفل/فتح الأرصدة لتنفيذ الصفقات.
- 📊 **محرك مطابقة أوامر** فائق السرعة مدعوم بـ Redis و Lua.
- 📈 **بث بيانات السوق** عبر WebSocket (أسعار، سجل الصفقات، دفتر الأوامر).
- 🛡️ **مراقبة أمنية وتسجيل تدقيقي** لجميع الإجراءات الحساسة.
- 🌐 **واجهة أمامية** حديثة بـ Next.js و Tailwind CSS.
- ☸️ **جاهز للإنتاج** مع مخطّطات Kubernetes و Terraform و CI/CD.
